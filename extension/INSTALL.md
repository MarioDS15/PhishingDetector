# Installation Guide

## Quick Install

1. Open Chrome and navigate to `chrome://extensions/`
2. Enable "Developer mode" (toggle in top-right corner)
3. Click "Load unpacked"
4. Select the `extension/` folder
5. Done! The extension is now active

## Test the Extension

Visit these test URLs:
- Safe: https://google.com
- Safe: https://github.com
- Test phishing patterns (educational): Create test URLs with suspicious patterns

## Troubleshooting

If the extension doesn't load:
1. Check that `extension/models/model_lite.json` exists
2. Open browser console (F12) and check for errors
3. Make sure all files are present in the extension folder
